# 🧬 生物奧林匹亞模擬複習系統 (IBO Prep Simulator)

一款針對**生物奧林匹亞（初賽 / 複賽）**的多方向模擬複習 PWA。目標：協助考生穩定達到**複賽前段**水準。

> 完全離線可用、資料儲存於本機（不上傳雲端）、可安裝為手機／桌面 App。

## ✨ 功能

| 功能 | 說明 |
| --- | --- |
| 📝 **模擬測驗** | 依「初賽 / 複賽」情境配題，作答後即時顯示**詳解**與考點標籤。 |
| 🎯 **弱點強化（自適應出題）** | 依歷史表現加權弱點領域與未練習題目，自動出題。 |
| 📚 **題庫瀏覽** | 依領域、階段、關鍵字篩選，逐題查看詳解與概念。 |
| 📈 **考題趨勢分析** | 各年度領域出題分布、出題比例 vs 官方權重、**高頻考點預測 🔥**。 |
| 🎯 **成績動態分析** | 成績趨勢曲線、領域掌握度雷達圖、各領域數據與**複習建議**。 |
| ⏳ **考試倒數** | 初賽 / 複賽日期倒數計時（可於設定調整）。 |
| 📱 **PWA** | Service Worker 離線快取、可安裝、響應式（手機 / 桌面）。 |

## 🧠 七大 IBO 理論領域（依官方權重）

細胞生物學 (20%)、植物解剖與生理 (15%)、動物解剖與生理 (25%)、動物行為 (5%)、
遺傳與演化 (20%)、生態學 (10%)、生物系統分類 (5%)。

## 📊 分析引擎（核心邏輯，皆有單元測試）

- **掌握度 (mastery)**：近因加權正確率 × 樣本信賴校正（避免少量僥倖即視為精熟）。
- **弱點偵測**：低於門檻或未練習的領域，依「掌握落差 × 官方權重」排序。
- **綜合準備度**：各領域掌握度依官方權重加權；達 **85** 視為複賽前段水準。
- **趨勢與高頻考點**：以「出現次數 × 近年熱度」計算熱度分數。
- **出題引擎**：標準配題（依權重）、自適應配題（加權弱點）、參數化變體生成（哈溫定律）。

## 🛠️ 技術棧

Vite · React 18 · TypeScript · Tailwind CSS · Zustand（localStorage 持久化）· Recharts · vite-plugin-pwa · Vitest。

## 🚀 開發

```bash
npm install
npm run dev        # 開發伺服器
npm run test       # 單元 + 整合測試 (Vitest)
npm run lint       # ESLint
npm run typecheck  # TypeScript 型別檢查
npm run build      # 產生 PWA 正式版到 dist/
npm run preview    # 預覽正式版
```
